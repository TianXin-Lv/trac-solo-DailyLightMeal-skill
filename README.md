# trac-solo-DailyLightMeal-skill
# 🥗 TRAE SOLO 参赛 Skill

## 一、技能基础信息

**技能名称：** 每日轻食菜谱推荐

**功能简介：** 一款桌面便利贴风格的轻食菜谱助手，温馨陪伴你每一天的饮食选择。记住你的口味偏好、智能匹配冰箱食材、贴心推荐减脂/增肌/控糖餐单——让你从"今天吃什么"的纠结中彻底解脱，轻轻松松做出美味又健康的简餐。

**适用场景：**
- 早餐/午餐不知道吃什么时，快速获取一份三明治或快手餐食谱
- 健身减脂期，想找低卡高蛋白的营养餐搭配方案
- 晚间饿了需要一份简易的营养加餐，不想大动干戈开火

**触发指令：**
> 「晚餐吃啥」「早餐吃啥」「吃什么」「菜谱」「食谱」「今天做什么菜」「三明治」「晚间加餐」「营养加餐」「按食材推荐」「冰箱里只有xxx」「减脂餐」「低卡餐」「增肌餐」「控糖餐」「帮我搭配」

## 三、导入&使用步骤

1. 点击上方文件名称，下载 `.skill` 格式文件到本地；
2. 打开 TRAE 官网，进入 SOLO 模块；
3. 找到「导入技能」功能，选中下载好的文件完成导入；
4. 在对话输入框输入**触发指令**（见上文），即可正常使用本技能。

---

## 四、功能特点

✨ **智能记忆，懂你口味**
- 自动记录你的忌口、甜/咸/辣偏好和营养目标
- 收藏加分、跳过减分，越用推荐越精准

🧠 **冰箱食材智能匹配**
- 告诉 Skill 你冰箱里有什么，秒级推荐可做的菜
- 食材不够？贴心地给你替代方案（比如没有鸡胸肉可用豆腐代替）

🏷 **营养目标精准适配**
- 减脂 / 增肌 / 控糖三大模式，热量和蛋白质数据一目了然
- 推荐结果自动按你的目标排序，挑选更轻松

📋 **每道菜谱完整呈现**
- 食材清单（含易遗漏酱料标注）→ 分步制作 → 口味小贴士
- 内容精简，每道菜 3-6 步，单人制作无压力

🍳 **内置 15 道轻食菜谱**
- 覆盖三明治、快手炒菜、晚间营养加餐三大品类
- 每道菜标注热量（kcal）、蛋白质（g）、碳水（g），健康更有数
